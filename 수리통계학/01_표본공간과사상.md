# 1강. 표본공간과 사상

Def(1) 표본공간(Sample space)
통게적 실험에서 나타나는 모든 결과들의 집합을 
표본공간이라 하고 이를 S로 나타낸다.

예제 1) 
(1) 한개의 동전을 던져서 앞면이 나오면 다시 동전을 던지고
뒷면이 나오면 한개의 주사위를 던지는 실험
=> S = { HH, HT, T1, T2, T3, T4, T5, T6 }

(2) 한개의 동전을 3번 던지는 실험
=> S = { HHH, HHT, HTH, THH, HTT, THT, TTH, TTT }

(3) 한개의 동전을 3번 던질때 앞면이 나오는 횟수의 실험
=> S = { 0, 1, 2, 3 }

(4) 비누공장의 생산제품 중에서 세개의 제품을 추출하여 검사한뒤
불량품(F)과 정품(H)을 분류하는 경우
=> S = { HHH, HHF, HFH, FHH, HFF, FHF, FFH, FFF }

(5) 중심이 원점이고 반지름이 3인 원의 원주및 내부의 모든 점
=> S = { (x,y) | x^2 + y^2 <= 9 }


Def(2) 사상(Events)
: 사상은 표본공간의 부분집합이다.
S = { 0, 1, 2, 3 } => 사상갯수 = 2^4 = 16개

예제 2)
비누 완제품 중에서 3개를 추출하여 검사한 후 정품(T)과 불량품(F)으로
구별할 때 불량품의 갯수가 1개 이상인 사상은?

표본공간: S={TTT, TTF, TFT, FTT, TFF, FTF, FFT, FFF}
F의 갯수가 1개 이상인 사상: A={TTF, TFT, FTT, TFF, FTF, FFT, FFF}

Than(1) 사상들의 관계
(1) 전사상(total event)
=> 표본공간 S의 모든 원소를 포함하는 사상

(2) 공사상(null event)
=> 표본공간 S의 어떤 원소 포함하지 않는 사상

(3) 여사상(complementary event)
=> 표본공간 S의 사상E에 속하지않는 S의 모든 원소들의 집합인 사상

(4) 합사상(union event)
=> 두 사상 E와 F에 대하여 E 또는 F중 적어도 한쪽은 일어나는 사상

(5) 곱사상(intersection event)
=> 두 사상 E와 F에 대하여 E와 F가 동시에 일어나는 사상

(6) 배반사상(mutually exclusive event)
=> 두 사상 E와 F에 대하여 E와 F중 한쪽이 일어나면
다른쪽은 일어나지 않을 때 E와 F는 서로 배반이라 한다.

예제3)
한개의 주사위
S={1,2,3,4,5,6}, E={1,2,3}, F={1,3,5}

합사상: {1,2,3,5}
곱사상: {1,3}
여사상: {4,5,6}
총사상의 갯수 = 2^6= 64개
