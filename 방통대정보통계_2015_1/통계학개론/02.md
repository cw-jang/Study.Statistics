#2강 데이터의 그래프에 의한 요약

##1. 데이터의 종류와 자료정리 목적

원자료raw data
: 표본에서 조사된 처음자료

양적자료quantitve data
: 수의 크기로 조사된 자료(몸무게, 키, 나이)

질적자료qualitative data
: 특성에 의해 구분된 자료(성별, 종교) 수학적 연산을 할 수 없다.

###통계자료의 측정척도
명목척도nominal scale
> 남자, 여자, 매체유형, ...

서열척도ordinal scale
> 수의 크기는 없지만 방향은 있는것
> 예) 지능-상,중,하, 수입-상,중,하

구간척도
> 물가지수, 섭씨-화씨온도 

비율척도: 절대0을 측정할 수 있다
> 몸무게, 허리둘레, 세금액

##2.도수분포표frequency table
1. 자료의 개수를 세어 n으로 나타낸다
2. 최대값과 최소값을 찾아 범위를 구한다
3. 급의 수를 정한다
4. 급의 폭을 정한다
5. 급의 경계값과 중심값을 구한다
6. 도수분포용지를 준비하고 도수를 표시
