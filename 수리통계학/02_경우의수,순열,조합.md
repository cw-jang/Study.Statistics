##Thm(3) 순열Permuation
서로 다른 N개의 원소중에서 r개를 뽑아 나열하는 방법 수
nPr = n(n-1)x ... x (n-r) = n!/(n-r)! (비복원추출)
중복순열 = n^r(복원추출)

*동차순열: 예) aaa, bb 나열하는 방법의 수
> 5! / 3!2!

*원순열 = 직순열 / n = n! / n = (n-1)!

예제11) a,b,c,d,e중 3개를 뽑아 만들수 있는 단어?
비복원추출: 5P3 = 5x4x3 = 60
복원추출(중복허락): 5^3 = 125

예제12) 10장의 카드에서 3장을 뽑아 나열
(1) 비복원추출: 10P3 = 10x9x8 = 720
(2) 복원추출: 10^3 = 1000

예제13) 4개의 붉은깃발, 2개의 청색깃발 신호
6!/(4!2!) = 15


##Thm(4) 조합Combiation
서로 다른 n개에서 r개를 선택하는 방법수는 
nCr = nPr / r! = n! / r!(n-r)!
(괄호를 사용한 조합  표기법 숙지할것)

예제14) A사 제품 10개중 3개, B사 제품 5개중 2개
10!/3!7! x 5!/2!3!
> 1200
